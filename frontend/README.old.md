# MediTrack Frontend

## Overview
This is the frontend for MediTrack, built with React and Material UI (MUI).

## Features
- Responsive dashboard UI
- Authentication (login/signup/logout)
- Role-based access (admin/doctor/nurse)
- Patient list, search, and filter
- Patient detail view (medical history, vitals, prescriptions, visits)
- Add/Edit/Delete patient records
- Appointment scheduling
- File uploads
- Notifications
- Export/print patient records
- Analytics and charts

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```

The app will run on [http://localhost:3000](http://localhost:3000) by default.
