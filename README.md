## Overview
MediTrack is a full-stack web application for managing patient health records, appointments, and notifications with a modern, responsive UI. It supports role-based access for admins, doctors, and nurses.

## Tech Stack
- **Frontend:** React + Material UI (MUI)
- **Backend:** Node.js + Express
- **Database:** MongoDB

## Features
- User authentication (login/signup/logout)
- Role-based access (admin/doctor/nurse)
- Dashboard overview (stats, charts, recent activity)
- Patient list (search, filter, pagination)
- Patient detail view (medical history, vitals, prescriptions, visits)
- Add/Edit/Delete patient records
- Appointment scheduling
- File uploads (e.g., reports, scans)
- Notifications (for appointments, updates)
- Export/print patient records
- Responsive, modern UI
